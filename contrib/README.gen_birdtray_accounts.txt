Use this script to generate the Accounts in 'Accounts to monitor'

Help with Examples
==================
$ gen_birdtray_accounts -h

Use Cases
=========
o You have 'many' accounts to monitor
o You do not want to use the GUI
o You make extensive use of TBird folders and email filters
o You wish to automate the creation of the account population

  Note:  you'll need a script to stop/start birdtray around the
         execution of this script.

Platform
========
o Linux
o Windows - it may work using cygwin

Requirements
============
o birdtray must not be running if results are persisted to its config
  file
